Поскольку я не знал, что именно я должен продемонстрировать в данном проекте,
я реализовал два подхода:
1) Для редактируемых таблиц (товары, аптеки, склады, партии) я максимально пользовался 
   наследованием и полиморфизмом. Классы Goods, Drugstores, Warehouses и Parties имеют
   общего предка BaseTable, в котором и содержится основная часть их функциональности.
2) Показ остатков товара для заданной аптеки реализован классом GoodsDragstoreView,
   получающим данные из GoodsDragstoreDao. Т.е. здесь я разделил View и Model.
   Кстати, фразу задания "Вывести на экран весь список товаров и его количество в выбранной
   аптеке" можно понимать двояко. Я вывожу список ВСЕХ товаров и для каждого из них
   количество в указанной аптеке.
